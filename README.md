# Riffly - Premium Music Learning Platform

Riffly is a high-fidelity, interactive music learning marketplace designed to connect passionate students with world-class instructors. Built with a focus on practice-driven education, Riffly provides the tools and feedback loop necessary to master any instrument.

## 🚀 Key Features

- **Interactive Practice Tools**: Built-in premium **Metronome** and **Instrument Tuner** (supporting 6-string guitar, 5-string guitar, and ukulele) directly in the browser.
- **Microphone-Powered Tuning**: Real-time pitch detection using the Web Audio API to help students stay in tune.
- **Instructor Feedback Loop**: Students can upload their practice recordings and receive personalized, time-stamped feedback from experts.
- **Bilingual Interface**: Full localization support for English and Turkish.
- **Comprehensive Curriculum**: Structured video courses with downloadable materials (PDFs, Tabs).
- **Responsive Design**: A sleek, modern dark-mode interface that works beautifully across all devices.

## 🛠 Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Styling**: Vanilla CSS Modules with CSS Variables for a custom design system.
- **Icons**: [Lucide React](https://lucide.dev/)
- **Audio Processing**: Web Audio API (Pitch Detection & Metronome Engine)
- **State Management**: React Context API
- **Language**: TypeScript

## 🏁 Getting Started

### Prerequisites

- Node.js 18.x or higher
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/falcno/rhytutor-repo.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🎸 How to Use the Tools

The **Practice Bar** at the top of the site gives you instant access to:
- **Metronome**: Adjust BPM and time signatures to stay on beat.
- **Tuner**: Select your instrument and grant microphone access to start tuning. The visual gauge and note indicator will guide you to perfect pitch.
